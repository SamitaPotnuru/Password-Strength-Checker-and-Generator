# 🔐 Password Strength Checker & Generator

A **single-page web application** that checks the strength of a password in real time and allows generating secure passwords.

---

## 🚀 Features

- ✅ **Real-time Password Strength Checker**
- ✅ **Weak / Medium / Strong** indicator
- ✅ **Strength Progress Bar**
- ✅ **Password Requirements Validation** (length, uppercase, lowercase, number, special character)
- ✅ **Show / Hide Password**
- ✅ **Generate Random Password**
- ✅ **Copy Password to Clipboard**
- ✅ **Modern Gradient UI**

---

## 💻 Technologies Used

- HTML  
- CSS  
- JavaScript  

No frameworks required – pure frontend code.

---

## 🖥️ How It Works

1. **User Types a Password**  
   - The app checks the password against rules using **Regular Expressions**.  
   - It updates the **strength bar** and shows which rules are satisfied.

2. **Generate Password**  
   - Click **Generate** to create a **random strong password**.  
   - Automatically fills the input box and checks its strength.

3. **Copy Password**  
   - Click **Copy** to copy the password to clipboard for easy use.

4. **Show/Hide Password**  
   - Toggle visibility of your password with the **Show button**.
